## Что это и откуда взялось

На входе был **дизайн-эталон из Claude Design** — «bundled» HTML (`3D-Scan NN (готовый
файл).html`, ~4 МБ): реальная вёрстка лежала JSON-строкой в `<script type="__bundler/template">`,
а картинки (16 шт.) и шрифты (24 woff2) — base64 в манифесте. Это был визуальный образец,
а не рабочий сайт: интерактив держался на проприетарном рантайме Claude Design (`<x-dc>`,
`DCLogic`, `sc-for`, `{{ }}`-биндинги, JS-бандл).

До этого в репозитории жил **другой, старый дизайн** (бежево-зелёная «бумажная» тема, Onest).
Его заменили новым графитовым прототипом 1:1, сохранив инфраструктуру публикации.

## Архитектура (ключевое решение)

Прототип перенесён в **чистую статику: HTML + CSS + vanilla JS, без рантайма Claude Design**.
Вся видимая вёрстка — обычный HTML с инлайн-стилями (переносится пиксель-в-пиксель). На
рантайме висели только калькулятор и аккордеон FAQ — переписаны на ~40 строк ванильного JS.
Сборки нет, GitHub Pages отдаёт статику напрямую.

Дизайн-токены: фон `#0E1219`/`#14181F`, синий `#0B5FFF`, оранжевый CTA `#FF5A1F`;
шрифты Space Grotesk / Manrope / IBM Plex Mono (self-hosted woff2).

## Структура

```
index.html            — вся страница (вёрстка + CSS в <head> + vanilla JS внизу)
privacy.html          — политика конфиденциальности (152-ФЗ)
robots.txt, sitemap.xml
assets/img/*.webp      — 16 изображений (вынесены из base64, сжаты в WebP)
assets/fonts/*.woff2   — 24 шрифта (self-host)
CNAME, favicon.svg/png, apple-touch-icon.png
```

## Калькулятор (Вариант B — детальный)

Перенесён **из прежней версии сайта** (старый `index.html`) со всеми реальными ставками,
визуально пересобран в графитовом стиле. 5 вкладок, расчёт обновляется на лету:
- **Помещения:** ставки 60–260 ₽/м² по типу результата (скан ч/б/цвет, план, +инж.сети,
  3D, Archicad), мин. заказы для ≤100 м², LOD200/300 (×1.5), НДС 0/5/25 %, км×200 ₽,
  скидки за объём 1000+ м² (10/25/50 %), ОКН ×1.35, срочность +20 %.
- **Фасады:** ставка = скан+чертёж+модель ₽/м² (+ортофото 30), мин. 50 000 ₽, автовышка
  +30 000, ОКН ×1.35.
- **Топо/ВЛС:** 20/25/35 тыс ₽/га; ВЛС мин. 300 000 ₽.
- **Резервуары:** база по объёму (30–60 тыс ₽/шт) + документы, скидка 10+ шт.
- **Реверс:** база × коэффициент точности (1…4) + CAD.
Кнопки «Скопировать текст для КП» и «Отправить расчёт в заявку» (подставляет расчёт в форму).

## SEO / юр / адаптив

- `title`/`description`, Open Graph, canonical, schema.org `LocalBusiness` (ИНН, телефон,
  гео НН, услуги), `robots.txt`, `sitemap.xml`, один H1, фавиконки.
- `privacy.html` (152-ФЗ) + cookie-уведомление (localStorage).
- Адаптив: медиа-запросы через атрибут-селекторы `[style*="..."]` с `!important` —
  инлайн-гриды прототипа схлопываются на мобильных без правки разметки (десктоп 1:1).
- Картинки: base64 → WebP (**2.6 МБ → 0.97 МБ**, −63 %), `loading=lazy/eager`, осмысленные `alt`.
- Проверено Playwright: десктоп 1:1, мобильный без горизонтального скролла, консоль чистая.
